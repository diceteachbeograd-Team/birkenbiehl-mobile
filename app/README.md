# App Skeleton

Dieses Verzeichnis ist der vorbereitete Einstieg fuer die spaetere Flutter-App.

## Struktur

- `lib/core` fuer querschnittliche Logik
- `lib/features` fuer Feature-Slices
- `lib/platform` fuer plattformspezifische Adapter
- `lib/shared` fuer wiederverwendbare Bausteine
- `tests` fuer Test-Placeholder

Die Struktur ist bewusst code-nah, aber noch nicht durch Flutter CLI generiert.
