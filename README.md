# Birkenbiehl Mobile

Offline-first Sprachlern-App mit Fokus auf Birkenbihl-Methode.

## Fokus
- Sprechen und Verstehen bis B1
- Dynamische Übungen (anpassbar pro Lernprofil)
- Kinderfreundlich und inklusiv (Screenreader/Braille-kompatibel)
- Lokal nutzbar ohne Internet

## Modell
- Für Lernende dauerhaft kostenlos
- Freiwillige Spenden im Elternmodus
- B2B-Lebenslizenz für Schulen/Unternehmen
